Nome Descrição
color A cor da mensagem em código HEX
const message[] O conteúdo da mensagem a ser enviada para o todos os jogadores online

Exemplos:

    public OnPlayerConnect(playerid) {
    SendClientMessageToAll(0xFFFFFFAA, "Bem-vindos ao meu servidor!");
    return 1;
}


#define Vermelho 0xFF0000AA

    public OnPlayerConnect(playerid) {
        SendClientMessageToAll(Vermelho, "Olá, bem-vindos ao meu servidor!");
        return 1;
    }

Notas:

 Aviso

Evite usar especificadores de formato em suas mensagens sem formatar a string que é enviada. Isso resultará em falhas de outra forma.
